# Geometric Calculator
A program to draw and calculates geometric shapes.

### Currently Supports
- [x] Squares
- [x] Rectangles
- [ ] Triangles
- [ ] Other Polygons
- [x] Circles
- [x] Ellipes
- [ ] 3D Shapes
- [ ] Cubes
- [ ] Cuboids
- [ ] Triangular Prismes
- [ ] Spheres
- [ ] Cones
- [ ] Cylinders
- [x] Near Life-Like Measurements for shapes that have an area of a few sqcm
