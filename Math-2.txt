     *  การหาพื้นที่รูปทรง
	15. การหาพื้นที่ทรงปริซึม
		ปริซึม คือ ทรงสามมิติที่มีฐานทั้งสองเป็นรูปสี่เหลี่ยมที่เท่ากันทุกประการ ฐานทั้งคู่อยู่ในระนาบที่ขนานกัน และด้านข้างแต่ละด้านเป็น
			พื้นที่ฐานปริซึมรูปสี่เหลี่ยมผืนผ้า =  กว้าว  x  ยาว
			พื้นที่ฐานปริซึมรูปสี่เหลี่ยมจัตุรัส  =  ด้าน x  ด้าน
			พื้นที่ฐานปริซึมรูปสี่เหลี่ยมคางหมู = 1/2 x สูง x ผลบวกของด้านคู่ขนาน
			พื้นที่ผิวข้างของปริซึม = ความยาวเส้นรอบฐาน X ความสูง
			พื้นที่ผิวทั้งหมดของปริซึม = พื้นที่ผิวข้าง X พื้นที่หน้าตัดหัวท้าย
		พื้นที่ผิวของปริซึม เมื่อคลี่ผิวข้างของปริซึมใด ๆ พบว่า จะเกิดเป็นรูปสี่เหลี่ยมผืนผ้าที่มีความยาวเท่ากับเส้นรอบฐานและส่วน
			พื้นที่ผิวปริซึม  =   พื้นที่ฐานทั้งสอง  +  พื้นที่ผิวข้างปริซึม
	16. การหาพื้นที่ทรงพีระมิด
		 พีระมิด คือ ทรงสามมิติที่มีฐานเป็นรูปเหลี่ยมใด ๆ มียอดแหลม ซึ่งไม่อยู่ในระนาบเดียวกับฐานและหน้าทุกหน้าเป็นรูปสามเหลี่ยม
	ที่มีจุดยอดร่วมกันที่ยอดแหลม
			พื้นที่ผิวข้างของพีระมิด = 1/2 X ความยาวรอบฐาน X สูงเอียง
			  		  = พื้นที่ของหน้าทุกหน้ารวมกัน
			พื้นที่ผิวของพีระมิด = พื้นที่ผิวข้างของพีระมิด X พื้นที่ฐานของพีระมิด
	17.  การหาพื้นที่ทรงกรวย
		กรวย (cone) คือ ทรงสามมิติใด ๆ ที่มีฐานเป็นวงกลม มียอดแหลมที่ไม่อยู่บนระแนบเดียวกันกับฐาน และเส้นที่ต่อระหว่างจุดยอดและ
	จุดใด ๆ บนขอบของฐานเป็นส่วนของเส้นตรง
			พื้นที่ผิวข้างของกรวย  =  rl
			พื้นที่ผิวของกรวย =   rl  +  r^2 
				หรือ   =    (22/7 หรือ 3.14) X รัศมี X สูงเอียง + (22/7 หรือ 3.14) X รัศมียกกำลังสอง
	18.  การหาพื้นที่ทรงกระบอก
		ทรงกระบอก (Cylinder) คือ ทรงสามมิติใด ๆ ที่มีฐานเป็นรูปวงกลมที่เท่ากันทุกประการกับหน้าตัด และอยู่ในระนาบที่ขนานกัน 
	เมื่อตัดทรงสามมิตินี้ด้วยระนาบที่ขนานกับฐานแล้ว จะได้รอยตัดเป็นวงกลมที่เท่ากันทุกประการกับฐานเสมอ
			พื้นที่ผิวข้างของทรงกระบอก = ความยาวรอบฐานของวงกลม  x  ส่วนสูง
					        =  2 rh                 ,    ?  22/4   หรือ  3.14  
					หรือ  =  2(22/7 หรือ 3.14) X รัศมี X สูงตรง + 2(22/7 หรือ 3.14) X รัศมียกกำลัง 2
			พื้นที่ผิวของทรงกระบอก =  พื้นที่ผิวข้าง  +  พื้นที่หน้าตัดทั้งสอง
					   =   2 rh  +  2 r^2          ,    ?  22/4   หรือ  3.14  
	19.  การหาพื้นที่ทรงกลม
		พื้นผิวทรงกลม = 4r^2    หรือ  ( 4 X (22/7 หรือ 3.14) X รัศมียกกำลัง 2 )
	20.  การหาพื้นที่ทรงสี่เหลี่ยมมุมฉาก
		ทรงสี่เหลี่ยมมุมฉาก คือ รูปทรงสามมิติที่ทุกด้านเป็นรูปเหลี่ยมมุมฉาก และด้านตรงข้ามเท่ากันทุกประการและขนานกัน
			พื้นที่ผิวของทรงสี่เหลี่ยมมุมฉาก = ผลรวมของด้านทั้ง 6 ด้าน

:::  การหาปริมาตรรูปทรงเลขาคณิต  :::

	1. การหาปริมาตรของทรงกลม
		ในทางคณิตศาสตร์ให้ความหมายคำว่า ทรงกลม คือ  รูปเรขาคณิตสามมิติที่มีผิวโค้งเรียบ และจุดทุกจุดบนผิวโค้งอยู่ห่างจากจุดจุดหนึ่งเป็นระยะเท่ากัน เรียกว่า ทรงกลม
	จุดคงที่นั้นเรียกว่า จุดศูนย์กลางของทรงกลม
	ระยะที่เท่ากันนั้นเรียกว่า รัศมีของทรงกลม
		สูตรคำนวณต่างๆที่เกี่ยวข้องกับทรงกลม
			ปริมาตรทรงกลม = 4/3r^3    หรือ  ( 4/3 X (22/7 หรือ 3.14) X รัศมียกกำลัง 3 )  ,  เมื่อ r เป็นรัศมีของทรงกลม
	2. การหาปริมาตรทรงกระบอก
		ในทางคณิตศาสตร์ให้ความหมายคำว่า ทรงกระบอก คือ  รูปเรขาคณิตสามมิติที่มีฐานสองฐานเป็นรูปวงกลมที่เท่ากันทุกประการและอยุ่บนระนาบที่ขนานกัน 
	และเมื่อตัดรูปเรขาคณิตสามมิตินั้นด้วยระนาบที่ขนานกับฐานแล้ว จะได้หน้าตัดเป็นวงกลมที่เท่ากันทุกประการกันฐานเสมอ เรียกรูปเรขาคณิตสามมิตินั้นว่า ทรงกระบอก
		สูตรคำนวณต่างๆที่เกี่ยวกับทรงกระบอก
			ปริมาตรทรงกระบอก = r^2h
			ปริมาตรทรงกระบอกกลวง = h (R^2 - r^2)
			ปริมาตรทรงกระบอกตัน = a^2h   ,   เมื่อ a เป็นรัศมีของฐาน h เป็นความ สูงตามแนวดิ่ง
	3. การหาปริมาตรทรงกรวย
		ในทางคณิตศาสตร์ให้ความหมายคำว่า กรวย คือ  รูปเรขาคณิตสามมิติที่มีฐานเป็นรูปวงกลม มียอดแหลมที่ไม่อยู่ในระนาบเดียวกันกับฐาน และเส้นที่ต่อระหว่าง
	จุดยอดกับจุดใดๆ บนขอบของฐานเป็นส่วนของเส้นตรง เรียกรูปเรขาคณิตสามมิตนั้นว่า กรวย
		สูตรคำนวณต่างๆที่เกี่ยวข้องกับกรวย
			ปริมาตรทรงกรวย = 1/3 r^2h
			ปริมาตรทรงกรวยตัน =  1/3 r^2h          เมื่อ r เป็นรัศมีของฐาน h เป็น ความสูงตามแนวดิ่ง
	4. การหาปริมาตรทรงพีระมิด
		ในทางคณิตศาสตร์ ให้ความหมายคำว่า พีระมิด คือ  รูปเรขาคณิตสามมิติที่มีฐานเป็นรูปเหลี่ยมใดๆ มียอดแหลมที่ไม่อยุ่บนระนาบเดียวกันกับฐาน 
	และหน้าทุกหน้าเป็นรูปสามเหลี่ยมที่มีจุดยอดร่วมกันที่ยอดแหลมนั้น เรียกว่า พีระมิด
		สูตรคำนวณต่างๆที่เกี่ยวกับพีระมิด  คือ
			ปริมาตรทรงพีระมิด = 1/3 x พื้นที่ฐาน x สูง
	5. การหาปริมาตรทรงปริซึม
		ในทางคณิตศาสตร์ ให้ความหมายคำว่า ปริซึม คือ  รูปเรขาคณิตสามมิติที่มีฐานทั้งสองเป็นรูปเหลี่ยมที่เท่ากันทุกประการ ฐานทั้งสองอยู่บนระนาบเดียวกัน 
	และด้านข้างแต่ละด้านเป็นรูปสี่เหลี่ยมด้านขนาน หรือเรียกง่ายๆว่า แท่งเหลี่ยมตัน  
		สูตรคำนวณต่างๆที่เกี่ยวกับปริซึม คือ
			ปริมาตรทรงปริซึม = พื้นที่ฐาน x สูง
			ปริมาตรทรงปริซึมสี่เหลี่ยมผืนผ้า  =  ( กว้าง x ยาว )  x   สูง
			ปริมาตรทรงปริซึมสี่เหลี่ยมจัตุรัส  =  ( ด้าน x ด้าน )  x   สูง
			ปริมาตรทรงปริซึมสี่เหลี่ยมคางหมู  =  ( 1/2 x ผลบวกของด้านคู่ขนาน  x สูง )  x   สูง
			ปริมาตรทรงปริซึมสามเหลี่ยมด้านเท่า  =  x( ด้าน ) ^ 2 x  สูง
	6. การหาปริมาตรของทรงสี่เหลี่ยมมุมฉาก
		 ทรงสี่เหลี่ยมมุมฉาก คือ รูปทรงสามมิติที่ทุกด้านเป็นรูปเหลี่ยมมุมฉาก และด้านตรงข้ามเท่ากันทุกประการและขนานกัน
			ปริมาตรของทรงสี่เหลี่ยมมุมฉาก = ความกว้าง x ความยาว x ความสูง
			หรือ ปริมาตรของทรงสี่เหลี่ยมมุมฉาก = พื้นที่ฐาน x สูง
	7. การหาปริมาตรของทรงลูกบาศก์
			ปริมาตรของทรงลูกบาศก์   =   ( ด้าน )^3
